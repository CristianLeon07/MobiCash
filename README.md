# MobiCash
MobiCash, sera  una mini aplicación de banca digital creada como proyecto personal para practicar y demostrar arquitectura moderna en Android.
No es un banco real, no conecta con entidades financieras: todo está simulado, pero diseñado para sentirse como una app bancaria profesional.
A medida que ira avanzado el desarrollo, se iran subiendo porciones de codigo.
